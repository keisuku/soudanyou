# Definition Of Done

- [ ] `PLAN.md` updated for non-trivial work.
- [ ] Diff is narrow and repo conventions are preserved.
- [ ] Wine YAML changes were followed by `npm run wines:build`.
- [ ] Code/UI changes were checked with `npm run lint` and/or `npm run build`.
- [ ] Visual changes were inspected at desktop and mobile widths.
- [ ] Assumptions are written in `ASSUMPTIONS.md`.
- [ ] Result and validation are written in `CHANGELOG_LOCAL.md`.
- [ ] Reusable knowledge is stored in `memory/`.
