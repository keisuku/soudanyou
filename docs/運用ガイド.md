# 運用ガイド — ご近所ワインを日本一のサイトにするために

現状: GitHub Pages（静的のみ、AI機能は動かない）
目標: 独自ドメイン＋AIソムリエ稼働＋高速配信で「日本一のコスパワインサイト」

---

## 1. 独自ドメイン — 絶対取るべき

**理由**

- 信頼感（`keisuku.github.io/soudanyou` より `gokinjo-wine.jp` の方がブランドとして強い）
- SEO（ドメインオーソリティが独自ドメインに蓄積する）
- プラットフォーム非依存（Vercelから別サービスに移っても同じURL）
- 将来のメール（`info@gokinjo-wine.jp` 等）

**候補とコスト（年額）**

| TLD | 年額目安 | 日本での信頼感 | 備考 |
|-----|---------|--------------|------|
| `.jp` | **¥3,000〜4,500** | ◎ 最高 | 日本国内ユーザー向けに最適 |
| `.com` | **¥1,300〜2,000** | ○ | グローバル展開を見据えるなら |
| `.wine` | ¥5,000〜7,000 | △ | ドメインハック的、SEOは普通 |
| `.jp.net` / `.tokyo` | ¥1,500〜3,000 | △ | 安いが認知低い |

**推奨: `.jp`**（年¥3,500程度）

**取得先おすすめ**

1. **お名前.com** — 国内シェア最大、初年度安いが更新料に注意
2. **Cloudflare Registrar** — 更新料が卸値で最安、ただし`.jp`は非対応（`.com` なら最安）
3. **Google Domains の後継 Squarespace Domains** — 料金透明

**ドメイン候補案**

- `gokinjo-wine.jp`
- `wine-konbini.jp`
- `soudan-wine.jp`（サイト内でAIソムリエに「相談」できるから）
- `wine-nearby.jp`

**運用で忘れがちな点**

- 自動更新を必ずONにする（失効=サイト死亡、数千円の節約で大事故）
- WHOIS情報公開代行を使う（個人情報が登録簿に載らない）

## 2. ホスティング選択 — 結論から

**推奨: Vercel Pro + 将来的に Cloudflare CDN 併用**

Next.js を作ったチーム直営なので動作が最適化されている。`main` にプッシュしたら数分で本番反映。プレビューデプロイ（ブランチごとに一意URL）が便利。

| サービス | 無料枠 | 有料プラン | 向き |
|---------|-------|-----------|------|
| **Vercel** | Hobby無料 / 帯域100GB | **Pro $20/mo** / 帯域1TB | ⭐ Next.jsの最適解。当サイトに最適 |
| Netlify | 無料 / 100GB | Pro $19/mo | Vercelとほぼ同等、少し機能劣る |
| Cloudflare Pages | 無料 / 無制限帯域 | Workers Paid $5/mo | 静的なら最強。API Routeは Workers 変換が必要で手間 |
| Render | 750h無料 | Starter $7/mo | Node が動く。Next.jsは非最適 |
| Fly.io | 少額無料 | 従量 $5〜 | コンテナ任意デプロイ、設定重い |
| 自前VPS（さくら/ConoHa） | なし | ¥640/mo〜 | 運用コスト＝時間、非推奨 |

### Vercel Hobby(無料) でどこまで行けるか

以下を超えると Pro 移行が必要:

- 帯域 100GB/月（画像が重いと10,000PV程度でアウト）
- Function実行時間 100GB-hours/月（AIソムリエ頻発だと厳しい）
- 商用利用不可（アドセンス等を入れたら即Pro必須）

つまり **広告や収益化をする瞬間に Pro ($20 = ¥3,000/mo) へ**。

## 4. xAI Grok APIのコスト管理

AIソムリエは **grok-2-1212** 使用。

- 入力: $2/M tokens（約¥300）
- 出力: $10/M tokens（約¥1,500）
- 1リクエスト ≒ 入力3,000トークン + 出力300トークン ≒ **$0.009（約¥1.4）**
- **10,000リクエスト/月 ≒ ¥14,000**

**コスト抑制の定石**

1. カタログを毎回40本に絞って渡す（現状実装済み ✅）
2. 同じ質問＋フィルタ条件をキャッシュ（Redis / Supabase KV、24h TTL推奨）
3. 無料ユーザーは1日3回制限、登録で10回、有料で無制限など段階化
4. プリセット質問（現6件）に合致した場合は予想回答を返す

キャッシュ導入で AI API コストは**半分以下**に落とせる。

## 5. 推奨移行手順（今すぐ→日本一）

### Step 1: 独自ドメイン取得（所要30分、¥3,500/年）
1. お名前.com または Cloudflare Registrar で `.jp` 取得
2. DNS を Cloudflare に委任（無料、最強のCDN/セキュリティ）

### Step 2: Vercel に移行（所要1時間、¥0〜3,000/月）
1. https://vercel.com で GitHub連携
2. `keisuku/soudanyou` をインポート
3. 環境変数を設定:
   - `XAI_API_KEY=xxx`
   - `NEXT_PUBLIC_SUPABASE_URL=xxx`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY=xxx`
   - `SUPABASE_SERVICE_ROLE_KEY=xxx`
   - `NEXT_PUBLIC_SITE_URL=https://gokinjo-wine.jp`
4. **`basePath: "/soudanyou"` を削除**（独自ドメインでは不要）
   - `next.config.ts` の `basePath` 行をコメントアウト
   - `AiSommelier` の `basePath` デフォルトも修正
5. Vercelダッシュボードでカスタムドメイン追加
6. Cloudflare DNS で Vercel の CNAME を向ける
7. HTTPS は自動発行

### Step 3: Supabase本番化（所要30分、¥0〜3,800/月）
1. 現状のSupabase接続を確認
2. `wines` テーブルへのマイグレーション実行（`supabase/` フォルダ参照）
3. Row Level Security を設定
4. Service Role Key は**必ずサーバー側のみ**で使用

### Step 4: アナリティクス導入（所要15分、¥0〜1,500/月）
- Vercel Analytics（無料で開始、$10/mo from 2,500 events/month）
- または **Plausible**（プライバシー配慮、$9/mo、GDPR準拠）
- Google Analytics はユーザー信頼性の観点でおすすめしない

### Step 5: Core Web Vitals モニタリング
- Vercel Speed Insights（Pro付属）
- PageSpeed Insights で月1チェック
- LCP < 2.5s、CLS < 0.1、INP < 200ms が目標

### Step 6: SEO追加施策
- Search Console 登録
- 主要ワインに「モンペラ」「アルパカ」等の検索語を `keywords` に
- 記事コンテンツ追加（`/guide/` 配下にワイン初心者ガイドを10本）
- 被リンク獲得（ワインインポーター、ブログ運営者）

## 6. 「日本一」の現実的な定義と競合

### 主要競合

| サイト | 強み | 弱み |
|-------|-----|------|
| **エノテカオンライン** | 品揃え、配送インフラ | コンビニワインは扱わない |
| **イカリ酒店 / リカマン** | 専門性 | 検索性・モバイルUXが古い |
| **ワインショップソムリエ** | SEO強い | スマホ最適化が甘い |
| **個人ブログ**（ワインおおはし等） | 濃い評論 | 検索導線が弱い |
| **当サイト** | コンビニ特化・AIソムリエ・コスパ指標・モバイル最適 | 認知ゼロ |

### 差別化の勝ち筋

1. **コンビニ・スーパー特化** — 既存大手がカバーしていないニッチ
2. **AIソムリエ** — 検索ではなく対話で選べる（当時点で競合ほぼなし）
3. **コスパ指標** — Vivino×価格の独自スコア
4. **モバイルUX** — サイト全体が親指操作前提

### 勝つために必要な追加施策（優先度順）

- [ ] 記事コンテンツ（月10本ペース、2年で240本）
- [ ] 楽天・Amazon アフィリエイト統合（収益化）
- [ ] 会員機能（お気に入り同期、レビュー投稿）
- [ ] モバイルアプリ（PWA→ネイティブ化）
- [ ] 他メディア露出（Togetter、はてなブックマーク、note）
- [ ] インフルエンサー連携（ソムリエ・飲酒系YouTuber）
